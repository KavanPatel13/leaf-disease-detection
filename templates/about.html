Project Overview

About Leaflence

Leaflence is a Flask-based leaf disease detection system that classifies apple leaf diseases using image classification with KNN, helping turn visual symptoms into quick machine learning predictions.

Project Description

Apple leaf disease classification with KNN

The project focuses on detecting apple leaf diseases from images using a machine learning pipeline. Users upload a leaf image, the system processes it, and the trained classifier predicts the likely disease category.

Problem

Why this system matters

  • Manual inspection is slow and can be inconsistent across large numbers of plants.
  • Visual disease symptoms may be subtle, making early detection difficult.
  • Machine learning provides a faster and more accurate digital screening approach.
Dataset

1092 Images

Performance

Accuracy: ~82%

The trained model delivers solid baseline performance for apple disease recognition and demonstrates the value of ML-assisted crop monitoring.

Tech Stack

Flask + OpenCV + Scikit-learn

The web application is built with Flask for routing and templates, OpenCV for image preprocessing, and Scikit-learn for the KNN classification workflow and evaluation pipeline.
